🔷 도형 4종

        
    
          
  • ▭ 과정 — 일반 작업·처리 (가장 많이 씁니다)
    • 
          
  • ▢ 시작/끝 — 흐름의 시작·종료·문서
    • 
          
  • ◇ 판단 — "예/아니오" 분기 (조건문)
    • 
          
  • ⬭ 데이터 — 결과물·산출물·파일
    • 
        

      


      

        
📝 어떻게 하나요? (10~15분)

        
    
          
  1. 닉네임 쓰고 시작 누르기
    2. 
          
  2. 왼쪽 도형 4종 중 하나 클릭 → 캔버스에 추가
    3. 
          
  3. 도형을 더블클릭해서 텍스트 입력
    4. 
          
  4. 또는 부서별 빠른 라벨에서 1클릭으로 도형+텍스트 추가
    5. 
          
  5. 도형을 마우스로 끌어서 위치 조정
    6. 
          
  6. 위쪽 🔗 연결모드 ON → 도형 A 클릭 → 도형 B 클릭 → 화살표 자동
    7. 
          
  7. 화살표 더블클릭 → "예/아니오" 같은 라벨 추가 (선택)
    8. 
          
  8. 우측 Mermaid 코드가 실시간으로 생성됨 → 📋 복사 → 🌐 mermaid.live 붙여넣기
    9. 
        

      


      

        
🚀 가장 빠른 사용법

        
    
          
  1. 본인 부서 탭 (예: 🏢 회계부) 클릭
    2. 
          
  2. 위에서부터 라벨 1클릭씩 → 7개 도형이 자동으로 캔버스에 깔림
    3. 
          
  3. 드래그로 위치 조정 + 화살표 연결
    4. 
          
  4. 📋 코드 복사 → mermaid.live 붙여넣기 → 끝
    5. 
        

        
💡 외울 필요 없어요. 5~8개 도형이면 충분합니다.
